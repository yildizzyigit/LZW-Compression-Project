# LZW Image Compression

This project implements the LZW (Lempel-Ziv-Welch) compression algorithm in Python for text and image files.

## Levels
- **Level 1** - Text file compression and decompression
- **Level 2** - Grayscale image compression
- **Level 3** - Grayscale image compression with difference encoding
- **Level 4** - Color image compression (RGB channels)
- **Level 5** - Color image compression with difference encoding
- **Level 6** - GUI application

## Requirements
- Python 3
- numpy
- Pillow
